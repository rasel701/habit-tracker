# Habit Hero

A smart habit tracking web application that helps you build positive daily routines.

## Live Site URL

[visit Habit Tracker ](https://dulcet-gecko-907bc8.netlify.app/)

## Features

- **User Authentication** — Login and Register using Firebase Authentication.
- **Add & Track Habits** — Easily create new habits and monitor your progress.
- **Streak Counter** — See how many days in a row you’ve completed your habits.
- **Progress Bar** — Visualize your growth over the last 30 days.
- **Public Habits Page** — Explore habits shared by other users.
- **Reminder Feature** — Set reminders for your daily habits.
- **Responsive Design** — Works smoothly on mobile, tablet, and desktop.

## Technologies Used

- React.js
- Tailwind CSS
- Firebase Authentication
- MongoDB & Express.js (for backend)
- Axios, React Router DOM, Toastify
